# **Turbowarp扩展**



#### 1.经纬度距离计算器.js

使用方法：

    输入两个点的经纬度即可计算两点的距离（米）









#### 2.中国地震烈度预估计算器.js

使用方法：

    输入震级，震源深度，震中距即可算出本地烈度

（使用双算法平均值）
